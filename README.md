# petalinux-ve2302-oob
Petalinux Repository for the Out of Box Design for the VE2302 SOM / Development Kit

Install PetaLinux and Vitis/Vivado 2024.2 tools on a compatible Linux distro.
Refer to the AMD Documentation for more details.

Source the appropriate setup scripts for both tools.

Enter 'make' in a console to build firmwares for the VE2302 Development Kit.
